# Jhelum River Heavy Metal Analysis - Code Repository

## Overview
This repository contains R code for the analysis presented in:
**"Bridging Source and Risk: A Combined PMF and Unsupervised Machine Learning Framework for Heavy Metal Management in a Transboundary Himalayan River"**

## Requirements
- R version 4.4.3 or higher
- Required packages:
  - FactoMineR (v2.11)
  - factoextra (v1.0.7)
  - corrplot (v0.92)
  - randomForest (v4.7-1.1)
  - ggplot2 (v3.5.0)
  - dplyr (v1.2.0)
  - tidyr (v1.3.0)
  - patchwork (v1.2.0)

## Installation
Run the following in R:
```r
install.packages(c("FactoMineR", "factoextra", "corrplot", "randomForest", 
                   "ggplot2", "dplyr", "tidyr", "patchwork"))
